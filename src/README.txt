Inicialmente deve-se indicar a ordem da matriz, uma vez que para a aplicação do 
método ela deve quadrada. 

Em seguida, os valores inteiros que compõem a matriz, linha a linha.  
Exemplo: Seja M a matriz:

A    B
C    D

Os valores devem ser inseridos na seguinte ordem: A, B, C, D. 

O resultado será a matriz otimizada para escolha das tarefas de modo que tragam 
menor custo.
